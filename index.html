Build Albums the Albumus Way: Structured, Tagged, Ready.

Albumus Audio Compiler empowers audio creators, archivists, and developers to generate polished music albums from raw audio using a clean, metadata-driven pipeline. No DAW bloat. No file chaos. Just organized, tagged, and cover-art-embedded output — ready to distribute.

Why Albumus Hits the Right Notes:

Automated Build Process

With a simple folder and metadata layout, Albumus will:

  • Convert any .wav, .flac, .mp3, and .ogg file into FLAC, MP3, OGG, and WAV formats
  • Auto-detect track numbers and format filenames
  • Embed metadata and album art from structured .json and folder.png
  • Generate custom-sized album art (cover.jpg, thumb.jpg, etc.)
  • Output organized per-artist, per-album folders for clean packaging

User-Friendly GUI

Manage your project with ease using our intuitive graphical interface, built with CustomTkinter, offering:

  • Directory management with recent project history for fast switching
  • One-click compile or cleanup
  • Visual task categorization and tooltip descriptions

Developer Control

  • All configuration handled via readable `.json` files
  • Set quality targets, formats, compression preferences, and art sizes
  • Customize FFmpeg behavior per format
  • Fully local — no telemetry, no cloud dependence

Light, Local, and Portable

Albumus runs on Windows and Linux. All dependencies are Python-based (plus FFmpeg), with:

  • No web stack
  • No Node.js
  • No bulky installers

Tools of the Curator

(Key Features)

Metadata with Precision

Per-song .json metadata for fine-grained tagging

Custom Album Art Sizes

Automatic resizing from a single source image

Output in Multiple Formats

Generate industry-standard formats in one go

Logging and Audit Trail

Track every action with detailed log files saved to the _log/ folder. Whether you're troubleshooting or archiving your workflow, Albumus gives you a transparent record of everything it does.

Clean and Reset

Start fresh by deleting the entire out/ directory with one click or command. Useful for removing old or partial exports before a clean rebuild.

The Albumus Strategy

(How It Works)

  1. Structure Your Input:
    Point Albumus to your project directory and it will automatically configure itself.

  2. Compile and Tag:
    Compile albums or clear output with one click or command.

  3. Archive and Deliver:
    Organized, tagged, multi-format audio ready for publishing.

System Setup & Requirements

  • OS: Windows, Linux
  • Dependencies: Python, FFmpeg
  • Supported File Types: .wav, .flac, .mp3, .ogg
